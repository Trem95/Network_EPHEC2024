Hello, in order to take full advantage of the capabilities of this program, please follow the instructions below to the letter:

- Perform a full build of the solution
- Open 2 pages of file explorer and go to the bin/Debug folder of the NetworkReceiver and NetworkSender folders of the solution.
- In the folder containing the NetworkSender.exe file, create a text file that you name as you wish and fill it in.
- Enter the command ‘cmd’ in the url of the respective folder to reach the command prompt 

NetworkReceiver utilisation:
- NetworkReceiver.exe <Chosen port> <Name of text file to create+ extension> 

NetworkSender utilisation :
- NetworkSender.exe <IP address of NetworkReceiver> <Port of NetworkReceiver> <Name text file to send + extension>

- To ensure that everything runs smoothly, launch the receiver, which will listen on all the ports, and then the sender to send your file.
- At the end, the programs shut down and you regain control of the command prompt.
